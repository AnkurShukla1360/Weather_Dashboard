# Weather Dashboard

A simple, easy-to-use application that allows users to check current weather conditions for various locations. 

## Features

* **Real-time Weather Data:** Display up-to-date temperature.
* **City Search:** Quickly search for weather conditions by entering a city name.
* **Responsive Design:** Clean interface that works well.

## Technologies Used

* HTML 
* CSS
* JavaScript
* Weather API 

## Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

You will need a free API key from a weather data provider to fetch the live weather information.
